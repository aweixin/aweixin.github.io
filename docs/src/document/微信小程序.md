## new Date

> 微信小程序 new Date() 方法在iOS设备上无效的问题的解决方法

```javascript
let date = "2021-01-04 00:00"
let now = new Date(date.replace(/-/g, '/'))
```

## 小程序分包

> 小程序的限制

-
    1. 分包不能引用主包中的第三方库

-
    2. 正常情况下，小程序对代码包的限制是：大小不超过 2M，超过这个大小无法预览，也无法上传代码

-
    3. 做了分包加载的情况下，限制是：每个包大小不超过 2M，所有包总共的大小不超过 16M。
